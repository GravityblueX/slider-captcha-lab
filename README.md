# slider-captcha-lab

A compliant local lab for studying human-like mouse trajectories and slider interactions on a local demo page.

This repository does NOT target or bypass any real website, paywall, CAPTCHA, or access control system.

Expected modules:

- local slider demo page
- human-like trajectory generator
- Playwright local drag demo
- CSV export for trajectory analysis

Research inspiration:

- https://github.com/riflosnake/HumanCursor
- https://github.com/topics/human-cursor
