# slider-captcha-lab v0.5.0

This release improves authorized deep-page diagnostics.

## Added

- Deep page map output in `src/page_probe.py`.
  - `depth` and `parent_index` for every frame.
  - `frame_chain` and `target_hint` fields for translating observations into authorized profiles.
  - Per-candidate `score` and `reasons`.
  - `summary.best_candidates` for manually reviewing likely controls.
- `docs/deep-page-map.md`.
- Report Center candidate table for page probe JSON files.

## Improved

- Regression checks now assert that page probe output includes best candidate hints.
- Smoke checks now require the deep page map documentation.
- README page-probe section now explains the new frame and candidate fields.

## Verification

- `python scripts/smoke_check.py`
- `python scripts/regression_check.py`
- `python scripts/cdp_attach_smoke.py`

## Safety

This release remains limited to local, owned, or explicitly authorized QA diagnostics.
It does not implement CAPTCHA solving or third-party anti-bot bypass logic.
