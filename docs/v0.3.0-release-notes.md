# 留痕 v0.3.0

主题：CDP 授权诊断与安全报告增强。

## 新增

- 新增 `src/cdp_diagnostics.py`：
  - 采集 Chromium CDP session 诊断；
  - 输出浏览器版本、target 摘要、frame 摘要、运行时环境字段；
  - Cookie 只记录名称和数量，不记录值；
  - 输出 JSON 可导入综合报告中心。
- 新增 `docs/cdp-authorized-diagnostics.md`，说明 CDP 诊断使用方式和安全边界。
- 新增 `docs/lux-net-dev-reference.md`，记录对 `tmzncty/lux_net` dev 分支的工程化参考点。

## 改进

- 综合报告中心可以识别 `cdp_diagnostics` JSON。
- `scripts/smoke_check.py` 覆盖新增 CDP 诊断文件和文档。
- `scripts/regression_check.py` 增加本地授权 demo 的 CDP 诊断回归检查。
- README 增加 CDP 授权诊断章节。

## 安全边界

本版本不实现验证码自动求解，不提供真实网站风控绕过逻辑，不输出 Cookie 值。
CDP 能力仅用于本地、自有或明确授权页面的 QA、诊断和报告留档。

## 验证

```bash
python scripts/smoke_check.py
python scripts/regression_check.py
python src/cdp_diagnostics.py examples/authorized_deep_page_profile.json --headless --out cdp-diagnostics-result.json
```
